圖文版本：http://heaven2013.pixnet.net/blog/post/280383529
我們來到歐舒丹L'OCCITANECaf
一入座,就被這裡的寬敞吸引
我很喜歡屋內的挑高設計
沒有壓迫感
在室內可以舒適的聊天、用餐
//
不脫離品牌的整體感
歐舒丹咖啡點餐本、桌墊
很一致性
//
輕食部分,愛麗絲點了庫克太太三明治,280元
老實說在點餐時,我看到菜單本的參考圖片
本人覺得非常不可思議
竟然只有一片吐司吶
這個吐司一定很特別,二話不說就點它
//
吐司內餡全是起司、火腿
再裹著香濃的白醬後焗烤
每一口全是紮實感
半熟蛋黃還是嵌入吐司內的
服務員建議,把蛋黃搓破後
蛋汁用來塗滿吐司表層
//
輕食餐點可升級套餐甜點+飲品,+250元
我點了草本舒緩花茶,超養生耶
人到歐舒丹來用餐
莫名的就覺得自己好像該點
花啊、草啊之類的茶,自以為的狀態
這個草本舒緩花茶,個人覺得還滿好喝的
很像身體SPA完會喝的茶
迷迭香+薰衣草
喝起來有涼涼的薄荷味
好像真的也讓身心放鬆
倘若喝不慣
同行姊妹點的咖啡歐蕾也不錯
滿大碗的,要拿碗裝
看,水杯放一旁
是否小巫見大巫
-
覆盆莓熔岩巧克力
一擺出來氣勢非凡,就是盤子特別長啦
溫熱的巧克力蛋糕
要搭配香草冰淇淋
真的非常非常美味
劃開後,覆盆莓巧克力醬也流出來
微微酸酸的口味
配上巧克力的甜
一百分吶,陶醉
歐舒丹烤布蕾
鎮店之寶吧
我看每一桌都有它的蹤跡
雖然草莓在我的鏡頭下拍的好乾
還是冰箱冰太久
至於歐舒丹烤布蕾,好吃與否
我覺得不難吃,但沒特別驚豔
//
檸檬塔
姊妹指名要的甜點
很特別的滋味
因為它非常酸
搭配外層酥皮讓酸甜中和一下
平常不愛吃酸食物的人
就不推薦這道甜點了,我自己是覺得味道不錯
//
