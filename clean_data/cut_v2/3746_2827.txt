[食記][台北泰式]暹廚泰式宮廷料理吉林店
Tue Nov 11 20:07:40 2014

圖文版:http://www.dtmsimon.com/2014/11/25218398.html

消費日期:2014/10

 ( http://goo.gl/OzG0HT )

隨著國人旅遊風氣興盛及越來越多新住民的加入，台灣現在要吃到一些
比較道地的異國料理機會似乎比從前更多了。對於一些東南亞方面的菜
式過去我們總停留在越南小吃、泰國小吃這類庶民吃的餐點，其實有些
以所謂的宮廷料理也就是更為精緻餐點為主題的店早就在台灣生根發芽，
像之前冰箱介紹過台中的很越南宮廷料理 ( http://goo.gl/52q7xT )還有本文介紹的暹
廚泰式宮廷
料理都是很具代表的例子。
 ( http://goo.gl/1gSt16 )

這回是朋友作東邀在暹廚吉林店，吉林店在哪邊其實很好找，大家看他
們對面現在是長這個樣子.....柯P的公民廣場對面，很好找是不是?是不是?

 ( http://goo.gl/myHDlH )

店們口立牌上有寫著一些優惠方案，但這些暫時與我無關因為這一攤是
朋友要請客的，有需要的朋友可以參考看看喔。

 ( http://goo.gl/j8Khdp )

走進店裡一看，裡面的裝潢真的跟我們一般吃泰國菜的店比很不一樣，
一整個就是透露出一股高貴的氛圍。不過這家店本來就是以太過宮廷料
理為訴求，主廚阿明師本身就是泰國人，同時也是泰王的御廚之一，聽
說有時候泰國有節慶還會常昂被徵召回國掌廚呢~另外值得一提的是阿明
師本身還獲得亞洲美食評鑑雜誌評選為台灣最佳前五名餐廳其中之一，
也就是說阿明師的泰國料理可謂全台第一喔，根本就是廚神來著~

 ( http://goo.gl/OvCK3O )

用餐區分為一樓跟二樓，二樓並設置了一個包廂。

 ( http://goo.gl/q9PtGs )

一上到二樓馬上透過窗戶看到柯P，啊柯P你離暹廚這麼近是有沒有進來
吃過飯啦?XD

 ( http://goo.gl/eKLZHx )

二樓有方桌也有圓桌，整個坐滿應該可以容納60人左右同時用餐吧!

 ( http://goo.gl/3bOqur )

而當晚唯一的包廂就留給我們這一群人使用囉~XD

因為是客隨主便，所以當天並沒有餐與點菜，我有餐點都由主人家張羅，
加上暹廚的菜色單價在網路上不是很好查到，所以本篇就不跟大家介紹
餐點價位，敬請見諒喔~

 ( http://goo.gl/aTCxbi )

炸蝦餅，這是東南亞泰國、印尼這邊很常見的小點心。

 ( http://goo.gl/hEx6wL )

暹廚特調奶茶，這是紅茶味比較重的泰式奶茶。

 ( http://goo.gl/H6Ma0b )

生菜捲類似越南春捲的做法，花生風味的酸甜醬汁是亮點。

 ( http://goo.gl/qA2GT5 )
http//goo.gl/3U1DuZ,魚露四季豆是採用清炒的方式,以魚路為主要風味來源,充滿濃濃的東南亞特有的味道
http//goo.gl/XEOX2T,泰式酸辣野菇,ˊ這道我在網路上的菜單找不到對應的菜名,基本上口味就是道地的泰式酸辣風味,袖珍菇這樣酸酸辣辣的風味吃起來真的很爽口
http//goo.gl/swThAK,這道是酸辣炸牡蠣,炸得香酥的大牡蠣配合店裡特製泰式酸辣醬汁以及九層塔一起食用,吃的到牡蠣的鮮甜,酸辣醬汁增加風味也順便有去油解膩的功效,http//goo.gl/IRZtSg,這是泰式脆皮雞,表層麵衣酥脆,裡頭雞肉依然鮮嫩多汁

http//goo.gl/oI5RyL,清蒸檸檬鱸魚,我想這道菜在這邊吃的話台灣應該沒有人敢跳出來跟暹廚比道地了吧,http//goo.gl/FUDeL5,咖哩南瓜牛肉,這道聽說是阿明師自己獨創的創意菜色,在別家不是阿明師管理的泰國餐廳可是吃不到的喔
http//goo.gl/t1UiXu,椰汁紅咖哩雞,這道是店裡的主廚推薦菜色之一
http//goo.gl/6Gt3Vv,這道應該是雞肉椰汁綠咖哩,口味跟紅咖哩完全不同,組成的元素也不同,http//goo.gl/gqMrkZ,金錢蝦餅在這邊阿明師把他設計成方便一人吃的尺寸,滾粉下鍋炸至金黃,單吃可以吃到鮮蝦滿滿的海味,如果沾點桂花蜂蜜醬會更可口喔,http//goo.gl/tKckkY,金黃酥脆的餅皮、豐富的手工鮮蝦內餡,讓人吃了之後真的會懷念不已,http//goo.gl/vHsQS8,這是打拋牛肉,所謂打拋就是九層塔啦,跟打跟拋的動作一點關係都沒有
這道牛肉我個人覺得很夠味,牛肉的熟度口感也剛剛好,這這天晚上我個人還滿喜歡的一道菜
http//goo.gl/fahpHC,香茅野菇湯是素食可用,取香茅的香與百菇的鮮煮出來的湯清澈鮮美
