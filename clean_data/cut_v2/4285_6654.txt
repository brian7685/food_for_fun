[食記]台北【甜點】和茗甘味處．抹茶控吃不完
Sun Apr 26 14:23:25 2015

   餐廳名稱：和茗甘味處
   消費時間：2015年/4月
   地址：台北市大安區大安路一段31巷40號
   電話：02-8771-7720
   營業時間：週日-週四1300-2200 週五-週六1300-2300
   每人平均價位：270
   有無包廂：無
   推薦菜色：抹茶千層蛋糕

   圖文版：http://heaven2013.pixnet.net/blog/post/279656965

抹茶醬汁淋在熱騰騰的鐵板上

滾燙的冒泡，鐵板也滋滋作響

今天愛麗絲要分享，台北東區巷弄內~

超級抹茶甜點 (如果你不喜歡抹茶，恐怕來這裡用餐會掃興~)

menu第一頁就是抹茶大集合 (店內鹹食，有提供抹茶之外的選擇啦)

不過抹茶千層派，只有星期五、六、日有提供

因此平日用餐的我們撲了空~

好可惜呀! (下回再戰)
和茗甘味處的用餐桌椅
極簡的色調,竹子和草綠色系的搭配
走的就是抹茶風嘛,誤
抹茶千層蛋糕,140元,+套餐,100元,=240元
//
甜度適中的抹茶千層蛋糕
嘗起來很爽口
不會讓人覺得膩
另外再升級成套餐,搭配小泡芙和霜淇淋
//
抹茶霜淇淋
甜中帶點淡淡的苦澀味
滿香醇的,但最後竟然沒吃完
大概是抹茶一次吃太多種類的關係
鐵板抹茶熱蛋糕,200元
嗯很甜
讓人最驚喜的片刻
就是淋上抹茶醬汁的瞬間
醬汁從冰淇淋上方,緩緩流到鐵板上
可以看到鐵板再吹著抹茶泡泡,你想像力也太扯了
//
熱蛋糕有包紅豆內餡
所以整份甜點,嘗起來真的太甜了
吃到最後很膩
因此,這次點的兩種甜點
我偏好抹茶千層蛋糕
//
和茗甘味處每人低消限制,蛋糕套餐、飲品、抹茶聖代,擇一
因為點餐的設計
並非每款蛋糕都能升級成為套餐
鐵板抹茶熱蛋糕、抹茶熔岩蛋糕
這兩者只能單點
為了符合低消
我們只好點杯飲料充數,個人覺得這樣的低消規定很不討喜
和茗甘味處,台北有兩間分店
愛麗絲前往的是忠孝店
