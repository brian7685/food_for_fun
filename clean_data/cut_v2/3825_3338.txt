[廣宣][台北美式][松山]ChiliCrabs美式餐廳
Fri Dec  5 17:29:55 2014

圖文版:http://www.dtmsimon.com/2014/12/chilicrab.html

消費日期:2014/11

 ( http://goo.gl/MsW7c1 )

沒有碗盤、沒有餐具，所有的食材在塑膠袋裡麵跟醬料攪和之後，整個
倒在桌面上，想大口咬龍蝦啃大螃蟹，沒別的方法，就麻煩各位直接用
手抓!

 ( http://goo.gl/XEDHge )

以上是現在美國加州最新流行的海鮮吃法，其實這兩、三個月已經悄悄
登陸台北囉，也就冰箱這篇文章要介紹的Chili Crabs 七哩蟹美式餐廳。
誰說到美式餐廳只能吃牛排喀漢堡?我們今天就大口吃蝦，盡情啃蟹吧!

 ( http://goo.gl/tE93mj )

這回冰箱很榮幸接到Walkerland x TaipeiWalker的邀約，一起跟著雜誌以
星級窩客的身分前往採訪七哩蟹美式餐廳。餐廳採用大量木作的設計，
入口往地下室的玄關處其實有一點讓人感覺來到了加州的碼頭邊。餐廳
入口處也設有兩個大又舒適的顧客等候區，排隊用餐也要坐得舒服待會
兒用餐才會開心不是嗎?

 ( http://goo.gl/g1SG96 )

地下室用餐區也是延續外面的風格，大量木作跟許多美式裝置藝術品讓
人有目不暇給，瞬間來到美國加州用餐的錯覺。

 ( http://goo.gl/Ty1e4E )

餐廳裡面另一個表現美式風格相當鮮明的地方就是到處都可以看到電視，
這個設計跟美國很多酒吧相當類似，也就是說球季一到，三五好友都能
約在一邊看球賽一邊吃海鮮、喝啤酒，真是一兼二顧，摸蜆仔兼洗褲!XD

 ( http://goo.gl/6zVsvx )

他們家的廁所外頭另外設置了一個長長的洗手台，全部不銹鋼打造帶有
濃濃的金屬感。因為我後來吃螃蟹沾得滿手來洗，突然想到這個設備應
該也是為了讓客人便於手部清潔的貼心設計，連水龍頭的開法都很不一
樣，開關就在水龍頭嘴的部位，以最小跟手部接觸面積兩個點就可以旋
開囉~因為印象深刻，臨走前特別折回拍了這張照片。

 ( http://goo.gl/rBVXVT )

 ( http://goo.gl/XyLWwy )
健康三色蔬菜棒+蘿蔓198元

大魚大肉前當然要先吃點蔬菜比較心安一點，這道生菜沙拉料理可以選
擇凱薩沙拉醬或是美式芥末油醋醬，兩種醬料來沾食蔬菜棒都很不錯。

 ( http://goo.gl/Ce8nVU )
美式炸雞翅178/6隻

 相當道地的美式雞翅炸法，吃辣的朋友可以搭配紐奧良香辣粉，不吃辣
的可以搭配檸檬椒鹽粉。

 ( http://goo.gl/6BaSB3 )

金黃酥脆的炸雞翅外酥內嫩，這時候旁邊來杯清涼的生啤酒一定更完美。

 ( http://goo.gl/MCm3f7 )

接下來就是今天的主角各式海鮮，如果選擇一樣以上海鮮就可以加點副
食，比如說甜玉米、綠花椰、杏鮑菇、秋葵、白馬鈴薯等等。以上所說
的副食品也能接受單點，每份都差不多是78元左右，細節請大家參考菜
單上面的資訊。

 ( http://goo.gl/xm0tQc )

所有選定的海鮮料及副食都會先被裝在這種透明的大塑膠袋哩，加入指
定的特製醬料出餐。別懷疑，這裡頭當然包括一整隻水煮熟成的波士頓
大龍蝦唷~不論食材多昂貴高檔，來到這邊通通都是裝在塑膠袋裡~~

 ( http://goo.gl/lxcncp )

再來就是桌邊服務，親切的服務人員會在桌邊大力晃動搓揉塑膠袋數次，
目的是要讓裡頭的醬汁與食材充分的拌勻。當辦得差不多時，服務人員
就會將手上大中的美味珍饈直接倒在預先鋪滿整個桌面的大張吸油紙上。

 ( http://goo.gl/AwQjMo )

沒錯，眼前這張桌就是個味這天的盤、這天的碗，真是超級粗曠的。

 ( http://goo.gl/n5VkX4 )

石蟹及白蝦都是以固定秤重作為計價方式，兩個都是台灣生產，石蟹1磅
388元、白蝦1磅428元。

 ( http://goo.gl/lUci2R )

這裡頭的白蝦其實不小隻，一開始我還以為是南美洲進口的進口貨，結
果後來詢問總經理Martin他告訴我都是台灣產的新鮮白蝦，真的讓我有
些意外，因為台灣產成本實在很高，像我去傳統市場買活白蝦1斤最少
也要喊到150元，那些還是個頭小的說。

 ( http://goo.gl/ZSJZhp )

再來另一個亮點不用說就是這尾波士頓大龍蝦囉~~~

 ( http://goo.gl/a8U1St )

只見Martin豪邁地把龍蝦提起，這個畫面真的是太邪惡了，過去我只在
自己美國朋友的臉書上看到，然後自己在電腦前恨得牙癢癢的XD

 ( http://goo.gl/ET6iKw )

龍蝦計價方式是以隻回單位，價格也是時價，請大家自行憲日詢問店家
牌價消費如何?

 ( http://goo.gl/pC6b0r )

一隻1.5磅的龍蝦個頭真的不小，開吃之前當然要跟好友先把他品頭論足
一番才算不虛此行拎公丟恩丟?

 ( http://goo.gl/uk6V4B )

面對這隻波士頓大個頭龍蝦擔心不知道要怎麼吃嗎?別擔心，在一切前置
作業(研究啦、拍照啦、打卡放毒啦等等)完成之餘，真的要開始享用，
店內工作人員會立馬過來桌邊為大家做方法講解及實際剝蝦蟹的服務。

 ( http://goo.gl/70jMvX )

貼心的桌邊服務，服務人員會幫客人重頭示範如何"支解"這隻龍蝦或螃
蟹，當然剝蝦子這部分我個人認為應該就不用店家示範了吧?

 ( http://goo.gl/8gQW7g )

整隻完整的新鮮波士頓大龍蝦螯，拿在手上真的太閃光了，吃在嘴裡蟹
肉破表的鮮甜度巧妙的跟沾在手指頭上醬汁融合在一起，微辣的口感，
長這麼大第一次覺得自己的手指頭 龍蝦的螯肉沾醬真好吃。

(長久以來我對這種新鮮蝦蟹都是以原味伺候，鮮少拌醬料一起食用，特
此說明!)

 ( http://goo.gl/dcv375 )

接著到了大人推薦的時間，今天味大家推薦七哩蟹的青蟹，也就是我們
熟悉的青蟳。每隻重量經篩選都在1~1.5磅之譜，個頭算是不小的，也是
以時價為計費基礎。

 ( http://goo.gl/aP98IZ )

蟹肉飽滿的大青蟹搭配來自路易斯安那州特殊風味的Cajun Seasoning醬汁，
蟹肉鮮甜彈牙，搭配醬汁微辣的口感超對味。

 ( http://goo.gl/JgiHZI )

雖然整晚剝蝦剝蟹剝得很累(這樣說好像會被唾棄)，不過望著桌上這座
用蝦殼、蟹殼堆起來的小山，心理的滿足感真的是不可言喻的，爽度
100趴喔~

秋天是吃蝦啃蟹最適合的季節，找天來趟七哩蟹，丟掉一切束縛豪邁的
吃一餐，放鬆靈魂也犒賞每天努力過生活的自己喔!
