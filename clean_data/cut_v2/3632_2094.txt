[食記]台北行天宮站漁六居食試營運初訪
Fri Oct 10 14:15:29 2014

圖文版: http://savvik123.pixnet.net/blog/post/185124843

店名：漁六居食
電話：試賣中暫無
地址：台北市松江路259巷14號
價格：600-1000 /人
用餐時間：2014/6
拍攝相機：Olympus XZ-1
個人評價：喜歡

小六吃多了，總會想要更多，於是乎變出了『小三』?! 不，是進階版的漁六
阿樹師傅 ( 跟風 ) 決定開一間新的店，提供更進一步不同的服務。目前還在試賣中
這天拜某前惡勢力的黑暗美少女之賜，揪了一團來吃  ( 很抱歉為了人身安全，這位的照
片不提供 )

光是走進來一看，整體裝璜的質感就是一整個不一樣

接下來就來介紹本日的800 NTD 無菜單料理

●沖繩水雲醋

去過很多次沖繩的本人，看到這菜就知道真的很專業啦
水雲，又稱海髮菜。它的親戚是昆布，屬褐藻類，沖繩的特產
之前去沖繩也被友人所托買過很多次

一般的水雲為了方便保存，是用鹽漬的。而漁六用的是新鮮品，而且是當季最佳期間取得
的
搭配醋十分好吃

●墨斗

可能是吧(笑) 介紹時說是軟絲小時候的樣子
但如果這樣寫，在當年上通識課時肯定會被教授給0分的 ( 軟絲是Neritic Squid ，螢烏
賊(脆管) 是 Sparkling enope squid ，但這個被稱為墨斗仔的，可能是耳烏賊
 Sepiolidae 不管是什麼，別想太多了 )

不過不重要，吃起來非常好吃

●軟絲胡麻

其實我不太喜歡吃煮熟的它( 沙西米多好吃....) 。不過這個調味的很不錯^^

●海鱺生魚片

只是上海鱺太平凡一點，所以一定有不同的招數，答案是......抹茶鹽

沾著吃與平常用醬油+山葵風味差很多
蠻特別的
要小心別沾太多........

●鮪魚大腹

從小六食堂開始到現在升級版的漁六，都有個傳聞是老闆如果去魚市看到太喜歡的魚，不
小心就會手滑買了下來
然後就會出現在我們的餐桌上
這個是一人一片

我記得第一次看到鮪魚大腹有多美味的說法，是來自於小時候......(對，小時候) 看的
漫畫 《GTO麻辣教師》
後來才陸續有機會得以親口嘗嘗

這像牛肉般的花紋以及超級肥美的油脂，完全只有超美味可以形容

當然，也有不識貨的人，比如說前幾天新聞在日本有個人吃鮪魚大腹然後和師傅抱怨說魚
太油，導致雙方翻臉，師傅拿刀出來追殺的case ....

●生魚片拼盤 鮭魚 , 白魽 , 紅魽

台灣人最愛的鮭魚，當然不在話下

其他兩種也都很新鮮好吃

●北海道冰燒

整顆的干貝以醬油與味增煮好，貝殼背面還簡單加了幾筆讓它看來更高檔些

這個Size , 口感一流沒話說

●烤天使蝦與味增蘿蔔

蘿蔔極佳
烤天使蝦在小六吃太多次了 XD

●烤綠竹筍佐海藻味增醬

雖然已不是綠竹筍最佳時節，但仍然非常鮮甜，味增醬調的甚好，與筍子很搭配

●星鰻柳川鍋

別看到柳川二字就以為這是九州料理
實際上它是標準的江戶料理，之所以叫柳川鍋，是因為最早作的店的店名叫『柳川』
......

但這也不是標準的柳川鍋，因為原始版是用泥鰍 ( 很多人不敢吃吧 ) 加上牛蒡與蛋作成

這邊以星鰻代替
個人覺得非常好吃 ( 但泥鰍版沒吃過，也許以後有機會去日本試試 )

●明太子雞翅

明太子味道不錯

●蕎麥冷麵

喜歡的麵條，因為用冷麵型態，Q度也夠

●味增湯

湯不錯，喜歡

●南瓜布丁

●總評

很棒的一餐，感覺在漁六吃的無菜單料理比起之前在小六吃的，更進一步。
阿樹師傅在食材與料理上的用心在菜色裡都顯現的出來
用餐此時尚在試營運

之後正式開放會再試試
