[食記][台中小吃][西區]台北小排飯
Tue Sep  9 00:07:44 2014

圖文版:http://www.dtmsimon.com/2014/09/porkrice39.html

消費日期:2014/09

 ( http://goo.gl/25LGiM )

每當遇到不知道要上哪裡吃飯的時候，我想買個便當回家我想一定是很
多朋友的首選。只是一般便當快餐店總是那幾樣固定菜色，如果有店能
提供消費者整面牆的菜色選擇，我想他們生意一定好到翻過去，就像本
文要介紹給大家的台北小排飯一樣。

 ( http://goo.gl/HXteui )

這家店雖然還滿夯的，但是事實上在網路發表過的食記其實沒有幾篇。
會知道這家是因為之前常常往返高鐵站跟家裡，在經過的路上看到的。
門口大大寫著招牌便當75元海報，真的讓人很難不印象深刻。

 ( http://goo.gl/vGije1 )

走進店裡，首先目光馬上被牆上大規格的餐點價目表吸引。每樣餐點都
有附照片，想吃什麼一看就很清楚，價位從75元到100元不等。

 ( http://goo.gl/YoH5gi )

找到位置之後不管是外帶或是內用都要先到櫃台點餐買單，然後再拿餐
券到旁邊的餐檯取餐。

 ( http://goo.gl/NHWalI )

這一鍋是招牌小排飯的料，因為當天太晚去所以已經賣完，剩下的湯如
果大家想喝可以跟服務人員說一聲，沒意外可以喝免費的。

 ( http://goo.gl/0AO3Ax )

這就是我們當天的晚餐-三寶飯跟爌肉飯。

 ( http://goo.gl/8OGbTQ )

白飯給得還滿大一碗，上頭的滷汁澆飯料還有碎菜脯可以自己在餐檯取用。

 ( http://goo.gl/vS1haK )

意外發現他們家的白飯竟然是混合圓米、長米一起煮的，真特別。

 ( http://goo.gl/che52H )

這是當天的例湯，也是因為太晚去沒啥料看不出來是什麼湯?

 ( http://goo.gl/zbC5VW )

這一碗就是剛剛那個小排骨的湯，因為本來要來吃的是這個主菜，既然
沒緣分吃到，喝個湯代表也不錯。排骨湯本身因為加入大量的蘿蔔、玉
米等蔬菜下去敖，所以味道意外的清爽，其實當湯喝也挺不賴的。

 ( http://goo.gl/Wlnoc1 )

配菜方面基本上都有四樣菜，如果在店用餐的話配菜打的份量會比便當
多很多，基本上郭吃這些配菜就很超值了，難怪每天在排隊。

接下來看看主菜部分.....

 ( http://goo.gl/bjaJHj )
爌肉飯75元

 ( http://goo.gl/yiSmQi )

採用的部位是豬五花肉這部分切成小塊滷製，顏色方面偏淺，沒有像我
們一般吃腿庫皮那種效果。

 ( http://goo.gl/GB1l16 )

基本上這味道就很像家裡媽媽會滷的那種魯肉，雖然看起來很油不過吃
起來的口感還好，肥肉部分已經充分滷透，入口即化是不誇張的形容。

 ( http://goo.gl/hxvxFr )
三寶飯85元

  ( http://goo.gl/fHXaU1 )

三寶飯顧名思義就是主菜有三樣，台北小排飯的三寶飯裡頭的三寶是炸
雞腿、炸蝦捲以及招牌炸肉排。雞腿看來應是醃漬好後沒有裹粉就下鍋
油炸，皮脆內嫩相當好吃。蝦捲跟肉排也都是店裡招牌，大家來這邊可
以點來吃吃看唷~
